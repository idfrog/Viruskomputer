Blog Timeline Virus Tentang

WORMS

Yaitu sebuah program destruktif yang menginjeksi sistem komputer dengan informasi dan mereplikasi diri, menyumbat sistem sehingga operasinya melambat atau berhenti.

EXE VIRUS

Sebuah virus tradisional, bisa salah satu bagian dari keseluruhan suatu program. Virus Executable ini akan berjalan ketika user mengklik programnya. Virusnya biasanya samar, karena memang bisa jadi tidak disadari. Jika sudah diklik akan mereplikasi program pada komputer.

Virus Bootsector

Virus ini tereksekusi ketika system boot. Virus bootsector ini terload ke dalam memory utama and menghancurkan data penyimpanan dalam hard disk.

Virus Makro

Virus ini biasanya berada dalam sebuah aplikasi, yang populer seperti berada dalam aplikasi word atau excell. Ketika kita membuka dokumen yang terinfeksi, virus makro akan terload kedalam memory dan menginfeksi bahkan menghancurkan data yang terdapat dalam hard disk.

Script Virus

Biasanya virus jenis script itu ditulis dengan menggunakan Visual Basic Scripting (VBS) dan bahasa pemerograman Javascript.

Virus Trojan

Virus ini adalah program penghancur. Trojan ini biasanya ada dalam program computer atau mungkin dalam sebuah crack games. Jika program di eksekusi, kemungkinan besar sistem operasi pada komputer bisa menjadi bermasalah.
